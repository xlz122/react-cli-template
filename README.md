## 简介

本项目是一个配置好route、redux的react cli模板。

## 项目使用技术栈

* React全家桶
* Redux
* TypeScript

## 项目说明

启动项目

```
yarn install
```

```
yarn start
```

项目打包

```
yarn build
```
